curl

curl is a versatile command-line tool used to transfer data to or from a server using various protocols like HTTP and FTP. It supports a wide range of options for requests and responses.

Options:

  • -I: Retrieves only the response headers from the server.
  • -o <file>: Saves the output to a specified file.
  • -X <command>: Specifies the request method (e.g., GET, POST).
  • -d <data>: Sends data in a POST request.
  • -H <header>: Adds a custom header to the request.
  • -k: Allows SSL connections without valid certificates.
  • -u <user:password>: Sets credentials for server authentication.
  • -L: Follows HTTP redirects.
