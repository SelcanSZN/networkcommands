traceroute

traceroute is a network diagnostic tool that tracks the path packets take to reach a target. It provides information about each hop along the route, helping diagnose connectivity issues.

Options:

  • -n: Displays only numeric addresses, skipping DNS resolution.
  • -m <max_ttl>: Sets the maximum number of hops (Time To Live).
  • -q <nqueries>: Specifies the number of queries sent per hop.
  • -w <wait_time>: Sets the time to wait for a response (in seconds).
  • -I: Uses ICMP (Internet Control Message Protocol) packets instead of UDP.
  • -T: Uses TCP SYN packets instead of UDP.
  • -p <port>: Sets the starting port number for queries.
  • -f <first_ttl>: Sets the initial TTL (Time To Live) value.
