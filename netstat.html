netstat

netstat is a powerful network tool that shows active network connections, network statistics, and routing tables. It effectively helps monitor and diagnose network-related issues.

Options:

  • -a: Displays all connections and listening ports.
  • -t: Shows only TCP (Transmission Control Protocol) connections.
  • -u: Shows only UDP (User Datagram Protocol) connections.
  • -n: Displays numerical addresses instead of names.
  • -p: Shows the program name and PID (Process ID) for each connection.
  • -r: Displays the current routing table.
  • -e: Provides more detailed information about each connection.
  • -s: Displays statistics for each protocol.
