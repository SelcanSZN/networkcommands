ifconfig

ifconfig is a command-line tool used for configuring and managing network interfaces on Unix-like systems. It allows viewing and modifying parameters such as IP addresses and network masks.

Options:

  • up: Activates a network interface.
  • down: Deactivates a network interface.
  • -a: Displays all interfaces, including inactive ones.
  • <interface> <address>: Assigns an IP address to a specific interface.
  • netmask <mask>: Sets the network mask for an interface.
  • broadcast <address>: Sets the broadcast address for an interface.
  • mtu <size>: Sets the maximum packet size.
  • promisc: Puts the interface in promiscuous mode to capture all packets.
